PORTAFOLIO WEB - ALEJANDRO ESTRADA ROMERO

Archivos incluidos:
- index.html
- escolares.html
- laborales.html
- por-que-y-para-que.html
- style.css

Cómo usarlo:
1. Abre el archivo index.html en tu navegador.
2. Para poner tu video:
   - Puedes insertar un <video> local o un iframe de YouTube.
3. Para poner tus fotos:
   - Cambia cada recuadro por una etiqueta <img>.
4. Si quieres subirlo a internet:
   - Puedes usar GitHub Pages, Netlify o Vercel.

Si quieres, en un siguiente paso te lo puedo dejar:
- con colores más elegantes,
- con animaciones,
- con tu video ya insertado,
- o con tus fotos ya acomodadas.
